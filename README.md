# XYZ Music Store

A basic music store webpage created using HTML.

## Features

- Displays music albums
- Shows album genres and prices
- Buy Now buttons
- View Cart and Checkout buttons
- Simple and beginner-friendly layout

## Technologies Used

- HTML5
